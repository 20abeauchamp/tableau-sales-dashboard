# Sales Performance Dashboard (Tableau)

## Overview
This Tableau dashboard analyzes business performance using sales and profit data. It provides interactive views of:

- Profit by state (geographic comparison)
- Sales by region and product category
- Sales trends over time
- KPI-style summary metrics

The dashboard is designed to support data-driven decision-making by highlighting high- and low-performing regions, category performance differences, and time-based trends.

## Key Questions Explored
- Which states and regions are most profitable?
- Which product categories drive the most sales?
- Are there noticeable trends over time?
- Where might there be opportunities for improvement?

## Tools Used
- Tableau
- Data visualization
- Business analytics
